# Dataser Overview

## About Dataset
This project uses a dataset sourced from Kaggle, which focuses on to identify Heart Disease with different clinical data like ECG, BP and etc.

It contains characteristics extracted from clinical data. The variables focus specifically on age, sex, chest pain, BP, cholesterol, fbs over 120, EKG, max HR, exercise angina ST depression, Slope of ST, Number of vessels fluro	and Thallium.
